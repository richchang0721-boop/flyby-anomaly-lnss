## 主假說：Perturbation Model（擾動模型）

**GPT 2026-06-27 提出的框架轉換——這是目前最成熟的物理圖像。**

### Source Model vs Perturbation Model

| | Source Model（舊）| Perturbation Model（新）|
|--|-----------------|----------------------|
| 場的來源 | 地球自轉**創造** LNSS | 地球自轉**擾動**背景場 |
| Helmholtz 的 S | 地球角動量 J | 地球對背景場的局部擾動 |
| 遠場邊界條件 | Ψ → 0（場消失）| Ψ → Ψ_bg（背景場均勻值）|
| CMB 方向的角色 | 需要額外假設 | 背景場的自然優先方向 |
| k² 的來源 | 未知（電漿失敗）| 背景場的自然曲率 |

### 物理圖像

類比：海洋（背景場）和船（地球）的關係。

```
宇宙背景場 Ψ_bg
（均勻，各向性由 CMB 決定）
        │
        ▼  地球質量 → 局部曲率擾動
           地球自轉 → 旋轉不對稱擾動
        │
        ▼  擾動場在球坐標展開
           → P₁ 模式（偶極，Anderson 公式）
           → P₂ 模式（四極，無風帶 ±54.7°）
        │
        ▼  太陽風（ap）壓縮局部擾動帶
           月球-太陽潮汐調制帶的形狀
        │
        ▼  Flyby anomaly
           = 太空船穿越局部擾動帶的路徑積分
```

### 為什麼這個模型更自然

**第一，解釋了 k² 的來源：**
在 Source Model 中，k² 需要外加（電漿、粒子質量等），都失敗了。
在 Perturbation Model 中，k² 是背景場在球坐標的自然波數——均勻背景場加上地球的球形邊界，自然給出特徵長度。

**第二，解釋了 Busack 的 CMB 方向：**
背景場不是各向同性的——地球相對 CMB 以 ~369 km/s 運動，這個運動在背景場中形成一個優先方向。Busack 的 C 參數（CMB 方向的不對稱項）自然地進入遠場邊界條件，不需要額外假設。

**第三，和卡門渦街類比更一致：**
卡門渦街中，流體（背景場）本來就存在，障礙物（地球）只是激發局部渦旋結構。「局部有序結構從背景場中湧現」比「地球憑空創造場」更自然。

### 重要警告

> ⚠️ **不要把「背景場」等同於「宇宙膨脹」或「暗能量」。**
>
> 這些都是過早的命名。目前只假設「存在某種大尺度背景場」，不指定其性質。
> 等模型和觀測成熟後，再問這個背景場對應到什麼已知的物理成分。

### 待確立的背景場性質

1. 它是否均勻（各向同性）？
2. 它的特徵長度是什麼？（決定 k 值）
3. 它和引力場如何耦合？（決定地球擾動的強度）
4. 它的方向性（CMB 方向）如何進入邊界條件？

### 類比：卡門渦街（Kármán Vortex Street）

> ⚠️ **Physical Analogy Only** — 以下類比僅用於建立直覺，不代表 LNSS 真的是流體渦旋，也不代表流體方程可以直接套用。

| 卡門渦街 | LNSS 類比 |
|---------|---------|
| 流體繞過障礙物 | 背景場繞過有質量、有自轉的天體 |
| 當 Re > 臨界值 → 渦街自發形成 | 當某個密度/角動量條件超過臨界值 → LNSS 自發形成 |
| 渦旋有固定的頻率和間距（Strouhal 數）| LNSS 帶有固定的特徵高度（B = 394 km？）|
| 渦旋不是「外加的」，是流體方程的自然解 | LNSS 不是「外加的場」，是背景場方程在特定條件下的自然解 |

### 為什麼這個類比重要

在這個框架下：
1. **不需要預設 LNSS 存在**——它是物理條件超過臨界值後的自發涌現
2. **P₁ 和 P₂ 的湧現**更自然——渦旋結構自然包含各種球諧模式
3. **場方程仍是 Helmholtz**——但不再是「描述外加場」，而是「描述自發結構的形成條件」
4. **臨界值概念**解釋了為什麼 LNSS 存在於特定高度範圍（宜居帶）：低於臨界高度（引力太強，背景場被壓制）；高於某高度（條件不再滿足，結構消失）

### 臨界條件的候選參數

| 參數 | 物理意義 | 估計量級 |
|------|---------|---------|
| 角動量密度 J/V | 單位體積的角動量 | ∝ ω·ρ，在帶邊界有特定值 |
| 能量密度 | 太陽風壓力 + 引力勢 + 旋轉動能 | 需要計算 |
| 特徵雷諾數類比 | 背景場「黏性」vs 旋轉力 | 待定義 |

## 背景場的物理候選（2026-06-28）

### 已知量和需要解釋的量

從 Anderson 係數反推，背景場 Ψ_bg 需要有：
```
波數：k = 1/B_main = 6.22×10⁻⁸ m⁻¹
等效密度：ρ_bg = ω_E²/(4πG) = 6.34 kg/m³
場波相速度：v_p = ω_E × B_main = 1,172 m/s
```

### 暗物質 + 等離子體複合介質假說

**物理圖像（Rich Chang 2026-06-28 提出）：**

```
宇宙背景（暗物質 + 暗能量）充滿整個空間
        │
        ▼  和地球附近的等離子體（電離層、磁層）相互作用
        │  可能的機制：引力場局部壓縮、磁場捕獲
        │
        ▼  產生某種複合介質（等效密度 ρ_bg，相速度 v_p）
        │
        ▼  這個介質的 Jeans 長度 = B_main
        │
        ▼  Helmholtz 方程的 k = 1/B_main
```

**計算約束：**

- 直接的暗能量效應：特徵長度在 Gpc 尺度，差 10⁴⁰ 倍
- 直接的局部暗物質：特徵長度 ~10⁶ km，差 100 倍
- 等離子體主導的耦合：等離子體頻率比暗物質頻率大 10⁸ 倍，暗物質修正可忽略

**「差 100 倍」的物理意義：**

局部暗物質密度（3×10⁻²² kg/m³）是最接近的候選，差距「只有」100 倍。如果存在讓暗物質在地球附近局部壓縮 ~100 倍的機制，就能填補這個差距。這個機制可能是：
- 地球磁場對帶電暗物質成分的捕獲
- 地球引力場對超輕暗物質玻色子的束縛（類比原子核束縛電子）
- 某種尚未識別的引力-電磁耦合

**警告（GPT 建議）：**
> ⚠️ 不要把「背景場」直接等同於「暗物質」或「暗能量」。先建立數學框架，確立背景場的性質，後問它對應什麼物理。
