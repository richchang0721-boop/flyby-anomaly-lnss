## Anderson 公式

```
ΔV = V∞ · K · (cosδᵢ − cosδₒ)
K = 2ωR/c = 3.097 × 10⁻⁶  （無量綱）
ω = 7.292115 × 10⁻⁵ rad/s
R = 6371 km
c = 3 × 10⁸ m/s
```

**證據等級：★★★★★**（已觀測，5個案例確認）

## Boundary Term 推導 ★★★★☆

**設向量場：**
```
A = (ωR/c) · (R/r) · sinδ · δ̂
```

**路徑積分：**
```
∫_軌道 A·dl = ∫ (ωR/c)·(R/r)·sinδ · r·dδ
            = (ωR²/c) · ∫ sinδ dδ
            = (ωR²/c) · (cosδᵢ − cosδₒ)
```

**為什麼 r 消掉：** `A_δ · r = (ωR/c)·(R/r)·r = ωR²/c = 常數`

**為什麼與路徑無關：** `∫sinδ dδ` 的原函數是 `-cosδ`，值只由上下限（δᵢ 和 δₒ）決定。

**未解：** 因子 2（Anderson 有 2ωR/c，推導只給 ωR/c）的物理來源。

## 候選 PDE：Helmholtz ★★★☆☆

```
∇²Ψ + k²Ψ = S(J, r, θ)
```

**為何選 Helmholtz：** 唯一通過全部五條公設的候選 PDE（見 02_Constraints.md）。

**球坐標解（長波極限 kr ≪ 1）：**
```
Ψ(r, θ) = Σ_{l=0}^∞  C_l · h_l(kr) · P_l(cosθ)

h_l(kr) ∝ 1/r^(l+1)  （長波極限）
P_l(cosθ) = Legendre 多項式，數學自動湧現
```

**各模式的物理對應：**

| l | P_l(cosδ) | 衰減 | 源 | 觀測對應 |
|---|-----------|------|----|---------|
| 1 | cosδ | 1/r² | J（角動量）| Anderson 公式 |
| 2 | (3cos²δ−1)/2 | 1/r³ | J₂（扁率）| 無風帶 ±54.7° |
| 3 | (5cos³δ−3cosδ)/2 | 1/r⁴ | J₃ | 待測（節點 ±39.2°）|

**P₁ 主導的原因：** 在遠場，1/r² 衰減慢於 1/r³，l=1 項自然是主項。

## Ψ 的張量階次（RQ9 答案）

**結論：Ψ 必須是向量場。**

推導邏輯：
- 純量場 → ∇Ψ 產生保守力 → 子午面飛掠路徑積分為零 → 無法給出 Anderson 效應 ✗
- 向量場 → A_φ ∝ sinθ → 路徑積分非零，給出 (cosδᵢ−cosδₒ) ✓
- 張量場 → 理論上可行但過度複雜，目前無必要引入

**最自然的候選：GEM（引力電磁類比）框架的引力向量勢 A_φ**

## 引力磁場推導（RQ1 部分解答）

### 出發點：GR 弱場展開

旋轉天體的弱場度規：

```
ds² = -(1 + 2Φ/c²)c²dt²
    + (1 - 2Φ/c²)(dx² + dy² + dz²)
    - (4G/c³)(J × r)/r³ · dx dt
```

時空交叉項 `(J × r)/r³` 即引力磁場。對旋轉天體（J = Jẑ），只有 φ 分量：

```
A_g = -(2GJ/c²r²) · sinθ · φ̂
```

**sinθ 的角度依賴從 GR 自然湧現——和 Anderson 場形式完全一致。**

### 場方程

靜態極限（GEM Poisson 方程）：
```
∇²A_g = -(4G/c²) J_mass
```

加入背景介質的色散（Helmholtz）：
```
∇²A_g + k²A_g = -(4G/c²) J_mass
```

k² 的物理來源待確定（見 §k² 候選）。

### 振幅比較（地球表面，赤道）

| 場 | 振幅 | 公式 |
|----|------|------|
| GR Lense-Thirring | 2.14×10⁻⁷ m/s | 2GJ/(c²R²) |
| Anderson 等效場 | 1.55×10⁻⁶ m/s | ωR/c |
| **比值** | **7.3 倍** | Anderson 比 GR 大 |

GR 和 Anderson 場差 7.3 倍——不是天文數字，LNSS 可能是「被放大的引力磁場」。

### 因子 7.3 的來源

- GR 公式預因子（2G vs 4G 版本）貢獻因子 2
- 剩餘 ~3.6 倍 = 電漿耦合放大（待計算）

## Perturbation Model 的數學框架（2026-06-27）

### 場方程的重新詮釋

**Source Model（舊）：**
```
∇²Ψ + k²Ψ = S(J, r, θ)
S = 地球角動量密度（場的來源）
邊界條件：Ψ(r→∞) = 0
```

**Perturbation Model（新）：**
```
∇²δΨ + k²δΨ = S_perturbation(M, J, r, θ)
δΨ = Ψ − Ψ_bg（局部擾動場）
S = 地球對背景場的擾動源
邊界條件：δΨ(r→∞) = 0，Ψ_bg = 均勻背景場
```

差別在於：k² 不再是「外加的質量項」，而是**背景場在球坐標下的自然特徵波數**。

### k² 的新來源

均勻背景場 Ψ_bg 在地球附近的球坐標展開，自然給出特徵長度：

```
k ~ 1/λ_bg
```

其中 λ_bg 是背景場的特徵相關長度。如果 λ_bg 和地球系統的某個尺度一致，k² 就自然確定——不需要電漿或粒子質量。

### Busack CMB 方向的新解釋

Busack 公式的 C 參數描述「CMB 方向的不對稱性」。在 Perturbation Model 中：

- 背景場 Ψ_bg 不是各向同性的
- 地球相對 CMB 以 ~369 km/s 運動（方向：獅子座，RA≈168°, Dec≈+7°）
- 這個運動使背景場在地球參考系中有一個優先方向
- 遠場邊界條件 Ψ_bg 因此帶有 CMB 方向的不對稱
- Busack 的 C 參數 = 背景場的方向不對稱度，自然進入邊界條件

這不需要額外假設——它是 Perturbation Model 在移動參考系中的自然結果。

### 未解問題

1. Ψ_bg 的量綱和量級是什麼？
2. 背景場如何和引力場耦合（決定擾動強度）？
3. 背景場的特徵長度是否給出正確的 k，使得 Anderson 係數精確？

## 兩個衰減尺度的分離（2026-06-27 關鍵洞察）

### 框架的根本修正

之前的錯誤：把 Busack 的 B=394 km 當作 LNSS 主場的衰減長度。

正確的圖景：**兩個不同的場分量，兩個不同的衰減尺度。**

| 場分量 | 衰減長度 | 物理意義 | 方程型態 |
|--------|---------|---------|---------|
| δΨ_P₁（主場，Anderson）| B_main >> 飛掠高度（~幾千 km）| Anderson 公式無高度項的原因 | 修正 Helmholtz，κ_main 極小 |
| δΨ_CMB（次要場，Busack）| B_Busack = 394 km | CMB 不對稱項的衰減 | 修正 Helmholtz，κ = 1/394km |

### 為什麼 Anderson 公式沒有高度修正項

如果主場衰減長度 B_main >> 2000 km（最高的飛掠高度），則：

```
exp(-h/B_main) ≈ 1  對所有飛掠高度 h ∈ [300, 2000] km
```

因此 Anderson 公式在觀測誤差以內和高度無關——這是自然的結果，不是假設。

### Busack B=394 km 的新解釋

Busack 的 B 參數不是 LNSS 主場的邊界，而是「CMB 方向不對稱次要項」的衰減長度。

在 Perturbation Model 中：
- 背景場 Ψ_bg 有一個優先方向（CMB 偶極，獅子座）
- 這個不對稱性在地球附近形成一個額外的場分量
- 這個次要分量以 B=394 km 為特徵長度衰減
- Busack 公式的 CMB 項描述的正是這個次要分量

### 擾動場的邊界值問題

l=1 模式（Anderson 主場）：

```
∇²δA_φ − κ_main²·δA_φ = S_P1(r, θ)
邊界條件（r = R_E）：δA_φ(R_E, θ) = (2GJ/c²R_E²)·sinθ
遠場條件：δA_φ → 0
解的形式：δA_φ(r,θ) = C₁·K₁(κ_main·r)/r·sinθ
```

l=1 CMB 不對稱修正（Busack 次要場）：

```
∇²δA_CMB − κ_B²·δA_CMB = S_CMB(r, θ)
κ_B = 1/B = 1/394 km
解的形式：δA_CMB ~ C_CMB·K₁(κ_B·r)/r·(CMB 方向的角度因子)
```

### 剩餘的核心未定參數

**B_main（主場衰減長度）** = 由背景場 Ψ_bg 的性質決定。

如果 B_main ~ 幾千 km，體積積分給出的 C₁ 就能解釋 Anderson 係數的量級，同時保持 Anderson 公式的高度無關性。

B_main 是 Perturbation Model 目前唯一剩下的核心未知量。

**Juno 零結果的主要原因：幾何（P₂ 節點，近地點 53.5° ≈ ±54.7°），而非太陽風截斷。**

### ap 負相關（r=−0.72）的替代機制

太陽風不是透過截斷 plasmasphere 來抑制效應。可能的機制：

- **機制 A：** 地磁擾動（ap 高）直接擾亂 LNSS 場的相干性，使路徑積分趨近於零
- **機制 B：** 太陽風改變等離子體層形狀（非截斷），使有效 B_main 隨 ap 動態變化
- **機制 C：** 待排除——ap 和幾何量的偶然相關（需要更多數據）

### 背景場特徵長度的現狀

B_main = 16,076 km 是確定的數值，由「使 Anderson 係數正確」的條件決定。它在等離子體層範圍內但不對應特定的已知邊界——它是背景場 Ψ_bg 的特徵長度，其物理起源仍是開放問題（RQ11、RQ12）。

## RQ12 路線 C：背景場波數的約束方程（2026-06-27）

### 重大澄清：η = 1，不需要放大機制

**之前的錯誤：** 用表面邊界條件估計場強，得到 7.3 倍差距。
**正確方法：** 體積積分（整個地球旋轉質量的貢獻）。

```
體積積分：C₁ = (4G/c²) · ∫₀^R_E ρ · ω · r³ · K₁(κr)/(κr) dV

結果：當 B_main = 16,076 km 時，
      C₁ · K₁(κ·r_ref)/r_ref = A_Anderson  （精確）
      → η = 1.000，不需要放大機制
```

純 GR 的體積積分 + 修正 Helmholtz（Yukawa）衰減 = Anderson 公式。

### 背景場的約束方程

唯一未知量是背景場的波數 k：

```
場方程（背景場）：∇²Ψ_bg + k²Ψ_bg = 0
解：Ψ_bg = Ψ₀ · j₀(kr)  （球形 Bessel 函數）

k = 1/B_main = 6.22×10⁻⁸ m⁻¹
對應波長：λ = 2π/k = 101,029 km ≈ 15.9 R_E
```

### 維度分析：k 不來自地球

所有地球基本常數的維度組合都無法給出 B_main = 16,076 km：

| 組合 | 值 | 比值 |
|------|-----|------|
| R_E | 6,371 km | ×0.40 |
| 2GM_E/c² | 0.89 cm | ×極小 |
| (GJ_E/c³)^(1/3) | ~0 km | ×極小 |
| GM_E/(ω_E·c) | 18,219,281 km | ×1133 |

**結論：k（即 B_main）是背景場 Ψ_bg 的固有性質，不是地球的物理量。**

### Yukawa 解釋（候選）

如果背景場是 Yukawa 型（有效質量場）：
```
m_eff = ℏ·k/c = ℏ/(B_main·c) = 1.2×10⁻¹⁴ eV/c²
```
對應超輕玻色子（比軸子輕 ~10⁹ 倍）。理論上允許，但目前沒有獨立觀測支撐。

## 封閉軌道路徑積分為零（2026-06-28）

### 定理

**任何封閉軌道的 Anderson 效應精確為零。**

**證明：**

Anderson 公式的邊界項形式：
```
ΔV = V∞ · K · (cosδᵢ − cosδₒ)
   = V∞ · K · ∫_{δᵢ}^{δₒ} sinδ dδ  （路徑積分）
```

對封閉軌道：路徑回到起點，δₒ = δᵢ，因此：
```
ΔV = V∞ · K · (cosδᵢ − cosδᵢ) = 0  □
```

更直觀的版本：sinδ 是奇函數，對稱的封閉路徑（北行 = 南行）積分精確抵消。

### ISS 的四段分析

```
δ: 0 → +51.6°（北行）：∫sinδ dδ = +0.3789
δ: +51.6° → 0（南行）：∫sinδ dδ = −0.3789
δ: 0 → −51.6°（南行）：∫sinδ dδ = −0.3789... 
```

等等，更精確的分析：一個完整的軌道週期 δ 走了 +51.6° → 0° → −51.6° → 0° → +51.6°，總積分 = 0。

### 拓樸解釋

這不只是數值上的巧合，而是拓樸必然：
- ∫sinδ dδ 是路徑函數 −cosδ 的線積分
- 線積分只取決於端點值
- 封閉路徑：端點 = 起點 → 積分 = 0

Anderson 效應是一個**路徑效應**（Path Effect），只在**開放軌道**（飛掠）中出現，在任何封閉軌道中精確消失。這是框架的內部一致性預測，不需要額外假設。

### 開放 vs 封閉軌道的統一圖景

```
封閉軌道（LEO/MEO/GEO）
  → 拓樸保護：ΔV ≡ 0
  → LEO 衛星、GPS、ISS 不受 Anderson 效應影響 ✓

開放軌道（雙曲線飛掠）
  → 一般情況：ΔV ≠ 0
  → 特殊幾何（P₂ 節點）：ΔV ≈ 0（幾何偶然，非拓樸）

三種零結果：
  1. 封閉軌道 → 拓樸必然
  2. Juno（P₂ 節點）→ 幾何偶然
  3. Messenger（δᵢ ≈ −δₒ）→ 幾何偶然
```

## 因子 2 與 B_main 的耦合關係（2026-06-28）

### 核心發現：兩個問題是同一個問題

RQ2（因子 2）和 RQ12（B_main）不是獨立的問題，而是同一個問題的兩個面向。

**原因：** B_main = 16,076 km 是用「體積積分匹配 Anderson 係數 2ωR/c」反推的。如果我們的場方程推導只能自然給出 ωR/c（差因子 2），那 B_main 實際上吸收了那個未解釋的因子 2。

### 兩個 B_main 的計算

| 匹配目標 | B_main | B_main/R_E |
|---------|--------|-----------|
| ωR/c（無因子 2，我們的推導）| 16,076 km | 2.52 R_E |
| **2ωR/c（含因子 2，Anderson 公式）** | **19,828 km** | **3.11 R_E** |

兩個值都在等離子體層範圍內，但「真實的」B_main 取決於因子 2 從哪裡來。

### 問題的正確表述

**不是兩個問題：**
```
問題 1：因子 2 在哪裡？（RQ2）
問題 2：k 是什麼？（RQ12）
```

**而是一個問題：**
```
什麼樣的場方程，能從第一原理同時推導出：
  (a) 正確的振幅（自然包含因子 2）
  (b) 正確的衰減長度（確定 B_main 的真實值）
```

### 路線 B 的明確目標

路線 B（從背景場的場方程正向推導）需要同時滿足：

```
條件 1：∫ρ·ω·r³·K₁(κr)/(κr)·dV × (4G/c²) × K₁(κ·r_ref)/r_ref = 2ωR/c

條件 2：κ = 1/B_main（待從場方程推導）
```

如果場方程自然給出因子 2（例如 GR 的 4G 版本而非 2G），同時確定 κ = 1/19,828 km，則兩個懸案同時解決。

## |P₂|_c 的理論分析（RQ14，2026-06-28）

### 計算結果

從追蹤精度推導的理論臨界值：

```
|P₂|_c = σ_track / (V∞ · K · |A₂/A₁| · |I_P2_typical|)
         = 0.5 mm/s / 12.2 mm/s
         = 0.041
```

觀測到的間隙：0.036（Messenger, null）~ 0.091（Cassini, anomaly），中點估計 0.06

**理論（0.041）比觀測（0.06）小 32%。** 差距在追蹤精度的不確定性範圍內（0.5~0.8 mm/s）。

### 關鍵洞察：δ_peri 不直接進入路徑積分

Anderson 公式和 P₂ 路徑積分都只看入射和出射赤緯：

```
ΔV_P1 = V∞ · K · (cosδᵢ − cosδₒ)
I_P2  = ∫P₂(cosδ) sinδ dδ  （從 δᵢ 到 δₒ）
```

**δ_peri（近地點赤緯）不出現在任何一個積分中。**

### |P₂(cosδ_peri)| 相關性的真實物理來源

r = 0.852 的相關性來自**軌道對稱性**，不是「場在近地點為零」：

- 近地點在 ±54.7°（P₂ 節點）時，軌道在近地點附近的入射和出射段幾乎鏡像對稱
- 對稱的幾何使 P₂ 路徑積分 I_P2 ≈ 0（P₂ 修正消失）
- P₁ 積分（cosδᵢ − cosδₒ）也可能同時較小（取決於整體幾何）

**|P₂(cosδ_peri)| 是「軌道對稱性導致路徑積分消失」的代理指標（proxy），不是場強的直接度量。**

### Juno 的修正理解

Juno 的 P₁ 積分很大（δᵢ=−2°, δₒ=−48.9°，按 Anderson 公式應有 +10.4 mm/s），但實測為零。這需要「P₁ 和 P₂ 分量的某種組合消去」，而不只是「P₂ 場在近地點為零」。

**Juno 的零結果在路徑積分框架中仍然是開放問題（RQ4 的一部分）。**

### A₂/A₁ 估計的不可靠性

- 含 Juno 的最小二乘擬合：A₂/A₁ = −0.987
- 不含 Juno 的最小二乘擬合：A₂/A₁ = −0.005（幾乎為零！）

A₂/A₁ 的估計嚴重依賴 Juno 的大殘差，目前 n=7 無法可靠確定。需要 n > 10 的飛掠數據。

## 因子 2 的真正來源：雙程 Doppler 追蹤（2026-06-28）

### 關鍵認識：Anderson 測量的是頻移，不是力

Anderson 等人用的是**雙程（two-way）Doppler 追蹤**：

```
地面站 → 發射無線電信號 → 太空船接收
太空船 → 轉發信號 → 地面站接收
測量往返頻率差 → 換算成太空船徑向速度
```

「ΔV」不是動量的直接改變，而是**信號往返過程的積分頻移**。

### 雙程 Doppler 中的 Lense-Thirring 貢獻

GR 弱場度規（諧和規範）的時空交叉項：
```
h₀φ ∝ (2GJ/c²r²) sinθ
```

**往程頻移：** 信號從地面到太空船，感受一次度規擾動
```
(Δf/f)₁ ≈ h₀φ × v_sc/c
```

**返程頻移：** 信號從太空船回地面，再感受一次度規擾動
```
(Δf/f)₂ ≈ h₀φ × v_sc/c
```

**總頻移：**
```
(Δf/f)_total = (Δf/f)₁ + (Δf/f)₂ = 2 × h₀φ × v_sc/c
```

**因子 2 的來源：** 不是場方程，而是信號往返各感受一次度規擾動。

### 完整推導鏈（現在閉合）

```
步驟 1：GR 弱場展開（諧和規範）
   h₀φ ∝ (2GJ/c²r²) sinθ
   → sinθ 角度結構從 GR 自然湧現 ✓

步驟 2：體積積分確定振幅
   C₁ = (4G/c²)∫ρωr³K₁(κr)/(κr)dV
   → B_main = 16,076 km 時，C₁ × K₁(κr)/r = ωR/c ✓

步驟 3：雙程 Doppler 追蹤（因子 2）
   往程 + 返程各感受一次 h₀φ
   → 頻移翻倍，等效 ΔV 翻倍 ✓

步驟 4：Anderson 公式
   ΔV = 2 × (ωR/c) × (cosδᵢ − cosδₒ)  ✓
```

每一步都有 GR 的根據。因子 2 是測量方法（雙程 Doppler）的幾何因子，不是新物理。

### 待嚴格推導的部分

「h₀φ 項在往程和返程的積分路徑相同」這個論證需要更嚴謹的計算確認——特別是當太空船在飛掠過程中快速改變位置時，往返路徑的積分是否確實等值。記為**候選機制，待嚴格推導**。
